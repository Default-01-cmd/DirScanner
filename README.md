## 🚀 How to use

### Prerequisites
Make sure you have Python 3 installed on your machine.

Installation
Simply clone the repository or download the `dirscanner.py` file:

#### commands
'-w' ex:= python3 dirscanner.py -w exemple.txt
'-t' ex:= python3 dirscanner.py -t 5 (default is 5)
